- [ ] Define zoning (optional) and post max counts per zone

## 4) Safety + Procedures (Baseline for All Units)

- [ ] Draft Makerspace Safety Handbook
- [ ] Post signage
  - [ ] PPE requirements
  - [ ] Machine rules
  - [ ] Emergency procedures
- [ ] Define PPE list (baseline + per-machine)
- [ ] Define onboarding requirement (no machine use until safety onboarding completed)
- [ ] Define incident reporting process + incident log
- [ ] Define out-of-service procedure (tag + lockout where applicable)

  - [ ] First aid kit stocked and accessible
